# CipherPixel 🔐

CipherPixel is a secure image steganography system that hides encrypted messages inside images.

## Features
- Encryption using key
- LSB image steganography
- Secure message transmission
- Simple web interface using Streamlit

## Technologies Used
- Python
- Streamlit
- Cryptography
- Pillow

## Usage
1. Upload an image
2. Enter secret message
3. Encode and share securely
4. Decode using the key
